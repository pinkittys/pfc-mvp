## **🚀 핵심 혁신 기술**

### **1. 맥락 기반 키워드 추출**
- **기존 방식**: 독립적인 키워드 추출
- **개선 방식**: 다른 디멘션 참조하여 연관성 있는 대안 생성
- **예시**: "사랑" + "생일" → "기쁨", "설렘" 대안

### **2. 스마트 추출 전략**
- **10자 미만**: 규칙 기반 (빠름, 낮은 정확도)
- **10-30자**: 간단한 LLM (중간 속도, 중간 정확도)
- **30자 이상**: 전체 LLM (느림, 높은 정확도)

### **3. 통합 WebSocket 시스템**
- **단일 연결**: WebSocket 하나로 모든 처리
- **실시간성**: 타이핑하면서 즉시 피드백
- **효율성**: API 호출 1번으로 모든 처리
